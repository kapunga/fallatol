Fallatól

Fallatól is a collection of micro libraries stemming from patterns I find myself using repeatedly for my personal projects and sometimes professionally. I decided to start collecting them all in one place so I could easily re-use them.

Please Note: Everything in here is very much a work in progress that is not ready for prime-time yet.

Libraries

Started

  • fallatol-config - A functional wrapper around sconfig.

Planned

  • fallatol-slack - Slack connector library
  • fallatol-llama - Ollama connector library

Name

Fallatól is a mashup for the Icelandic words 'fall' meaning a function in programming, and tól meaning tool(s). Icelandic is a rare enough language that using it in a library name is highly likely to avoid name collisions. Also, I'm part Icelandic and have started learning the language.
